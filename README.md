# Kashmir Sikh Connect

A static community website for Kashmiri Sikh students, graduates, professionals, and job seekers.

## Files

- `index.html` - website content and sections
- `styles.css` - responsive design and theme
- `script.js` - mobile menu behavior
- `netlify.toml` - Netlify deployment settings

## Logo

Place the logo in:

```text
assets/logo.jpeg
```

The header uses:

```html
<img class="brand-logo" src="assets/logo.jpeg" alt="Kashmir Sikh Connect logo">
```

## Deploy

Recommended first deployment: Netlify.

1. Create a Netlify account.
2. Drag this folder into Netlify Drop, or connect it to a GitHub repository.
3. Netlify will publish the site from the project root.
4. Forms submitted through the Join section will appear under Netlify Forms after the site is deployed.

For a custom domain, buy a domain from any registrar and point it to the Netlify site.
